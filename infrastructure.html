Projet Infrastructure Réseau - 2ème année

Présentation du Projet

Ce projet a pour objectif de mettre en place une infrastructure pour StationF. Nous avons configurés d'une infrastructure réseau complète avec serveur AD, HAProxy et gestion des switchs

Switch Cœur et Switchs d'Accès

Un switch cœur a été configuré pour gérer le routage entre les différents segments du réseau, assurant un flux de données optimisé et sécurisé. Les switchs d'accès sont utilisés pour connecter les utilisateurs finaux et appliquer les règles de VLAN pour segmenter le réseau selon les besoins de l'organisation.

Serveur Active Directory (AD)

Le serveur Active Directory a été configuré pour gérer les comptes utilisateurs et centraliser les droits d'accès sur le réseau. Cela inclut la création et la gestion des unités organisationnelles et des stratégies de groupe (GPO) pour sécuriser l'environnement.

Serveur HAProxy

HAProxy a été configuré comme un load balancer pour répartir les requêtes de manière équilibrée entre plusieurs serveurs web, améliorant ainsi la performance et la disponibilité de l'infrastructure réseau.

Projet réalisé par les étudiants en 2ème année de BTS SIO - Option SISR
