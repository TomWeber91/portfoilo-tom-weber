Projet Création de Site Web

Présentation du Projet

Ce projet avait pour objectif de créer un site web pour une startup, permettant de présenter ses services et de gérer les interactions clients. Le site est conçu avec HTML pour la structure, CSS pour le style, et PHP pour la logique côté serveur.

Structure HTML

La structure du site a été réalisée en HTML pour garantir une hiérarchie claire des éléments. Nous avons organisé le contenu en sections, articles, en-têtes et pieds de page pour une meilleure lisibilité et accessibilité.

Mise en forme avec CSS

Le design a été mis en place avec CSS pour apporter une identité visuelle cohérente. Nous avons utilisé des styles personnalisés pour les boutons, les sections et les titres, en appliquant des couleurs modernes et des ombrages pour une esthétique professionnelle.

Intégration de PHP pour la logique du site

PHP a été utilisé pour intégrer la logique de gestion des données du site. Cela inclut l'envoi de données vers une base de données MySQL, et la création de pages dynamiques pour améliorer l'expérience utilisateur.

Projet réalisé par les étudiants en 1ère année de BTS SIO - Option SISR
